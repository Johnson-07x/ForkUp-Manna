# Coding Rules

- Use Java 21.
- Use Spring Boot 3.
- Use Maven.
- Use MySQL.
- Use JWT Authentication.
- Use DTO pattern.
- No business logic in controllers.
- Use service layer.
- Use constructor injection only.
- Use Lombok.
- Use Global Exception Handler.
- Follow REST standards.
- Write production-grade code.
- Generate Swagger documentation.
- Generate unit tests.
