# Donor

As a donor,
I want to create a food donation,
so that surplus food can reach people in need.

Acceptance Criteria:

- Donor must be logged in.
- Donor can upload images.
- Donor can specify quantity.
- Donor can specify expiry time.
- Donation status defaults to AVAILABLE.

# Receiver

As a receiver,
I want to view nearby donations,
so that I can claim food quickly.

Acceptance Criteria:

- Show donations within configurable radius.
- Show quantity.
- Show donor location.
